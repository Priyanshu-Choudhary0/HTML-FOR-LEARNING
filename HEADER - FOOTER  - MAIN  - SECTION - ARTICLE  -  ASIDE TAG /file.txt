HEADER TAG -
PURPOSE : USED TO CONTAIN INTRODUCTORY CONTENT OR NAVIGATION LINKS

FOOTER TAG - 
PURPOSE : FOR FOTER CONTENT LIKE EXTRA INO OR LINKS

MAIN TAG -
PURPOSE : ENCLOSES THE PRIMARY CONTENT OF A WEBPAGE

SECTION TAG - 
PURPOSE : GROUPS RELATED CONTENT IN A DISTINCT SECTION

ARTICLE TAG - 
PURPOSE : ENCLOSES CONTENT THAT STANDS ALONE , LIKE A BLOG POST 

ASIDE TAG - 
PURPOSE : CONTAINS SIDEBAR OR SUPPLEMENTARY CONTENT
