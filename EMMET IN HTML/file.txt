EMMET - 

EMMET IS A SHORTHAND SYNTAX USED IN CODE EDITORS TO WRITE HTML FASTER . 
IT EXPANDS ABBREVATIONS INTO FULL CODE . 

<!DOCTYPE html>  //DEFINES THE HTML VERSION
<html lang="en">   // PARENT OF ALL HTML TAGS
  <head>   
   <title>MY FIRST WEBPAGE</title>  //TITLE OF WEB PAGE
  </head> 
  <body>   // PARENT OF CONTENT TAGS 
   <h1>NAMASTE BHARAT</h1>   HEADING TAG
  </body>
</html>
