TABLE TAGS IN HTML - 

<tr> TABLE ROW : USED TO DEFINE A ROW
<th> TABLE HEADER : USED TO DEFINE A HEADER CELLS , TEXT IS BOLD AND CENTERED BY DEFAULT
<td> TABLE DATA : THIS HOLDS THE ACTUAL DATA

CAPTION TAG -
PURPOSE : PROVIDE A TITLE OR DESCRIPTION FOR A TABLE 
MUST BE INSERTED IMMEDIATELY AFTER THE <table> OPENING
CENTERED ABOVE THE TABLE BY DEFAULT 

COL SPANS - 
ATTRIBUTE : USES THE COLSPAN ATTRIBUTE IN <td> OR <th> TAGS
PURPOSE : ALLOWS A CELL TO SPAN MULTIPLE COLUMNS HORIZONTALLY
