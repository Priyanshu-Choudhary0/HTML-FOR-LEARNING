ORDERED LISTS - 

PURPOSE : USED FOR LEARNING LISTS WITH HAVE A SPECIFIC ORDERED
ITEMS ARE AUTOMATICALLY NUMBERED
 <ol>
  <li> </li>
  <li> </li>
 </ol>

TYPES - 
> NUMERIC : DEFAULT TYPE [1,2,3,...]
 ATTRIBUTE : TYPE = "1"
> UPPPERCASE LETTER : [A,B,C,...]
 ATTRIBUTE : TYPE = "A"
> LOWERCASE LETTER : [a,b,c,...]
 ATTRIBUTE : TYPE = "a"
> UPPPERCASE ROMAN : [I,II,III,...]
 ATTRIBUTE : TYPE = "I"
> LOWERCASE ROMAN : [i,ii,iii,...]
 ATTRIBUTE : TYPE = "i"
